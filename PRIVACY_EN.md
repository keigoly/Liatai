# Privacy Policy

This Privacy Policy describes how user information is handled in the Chrome extension "Liatai" (hereinafter referred to as "this Extension").

## 1. Information Collection
This Extension does not collect users' personal information (name, address, email address, phone number, etc.).
Furthermore, users' browser viewing history, cookies, search history, etc., are never transmitted to the developer's server or third-party servers.

## 2. Data Storage and Usage
This Extension saves the following setting information in the local storage (Chrome Storage API) within the user's browser.

* **User Settings:** Language settings, auto-refresh interval, theme color, background mode, font size
* **Filtering Settings:** NG words and NG user IDs registered by the user

These data are used solely for the purpose of maintaining the functionality of the Extension and are never transmitted externally.

## 3. Access to External Services
This Extension accesses Yahoo! Real-time Search (`search.yahoo.co.jp`) to retrieve data and display it within the side panel.
This communication is performed directly from the user's browser, and the developer does not intermediate.

## 4. Disclaimer
This Extension is an unofficial tool developed by an individual. It has no relationship with Yahoo! JAPAN or LINE Yahoo Corporation.
The developer assumes no responsibility for any damages resulting from the use of this Extension.
